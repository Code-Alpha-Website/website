Creating a new tenant

When Anudan database is setup it also creates a single user with administrative privileges called the Platform Administrator (PA).

The PA is resonsible for creating the first tenant (the Grant giving organization). Please see the steps below to create a new tenant.

  1. Launch http://localhost:8080 on your browser. And enter PA credentials as administrator@anudan.com / password

  2. In the tenant setup screen provide details as given below and click

    1. Tenant: Name of the Grant giving organization
    2. Tenant Slug: One-Word short code for the organization
    3. Send invite to: Email of the person who will the tenant administrator
    4. Choose File: Organization logo image (png recommended)

    3. Creating a new tenant carries out the following activities

    • Stores uploaded logo
    • Creates a Granter organization with url http://<short-code>/localhost:8080
    • Creates user with email set in Send invite to. An invitation email is sent out with a link to verify email and complete the registration

Tenant Administration

The tenant user created in the previous step is by default the Tenant Administrator (TA) and is responsible for the configration and upkeep of the tenant organization
