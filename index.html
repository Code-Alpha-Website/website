Code Alpha is an open source initiative towards building societal platforms and tools in the development sector. The focus is on building affordable, accessible & usable software solutions that solve real problems for users esp. in areas with technology gaps or where existing solutions are seen as not affordable. Key principles that drive the choices in design, development & infrastructure include (but not limited to) scalability, performance, security, accessibility and availability appropriate to the solution space.

Code Alpha is an initiative of Social Alpha.
